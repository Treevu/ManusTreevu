# Executive Dashboard - User Guide

## Overview

The Executive Dashboard provides C-suite executives and administrators with real-time insights into employee financial wellness, churn risk management, and intervention effectiveness. This guide covers all features and how to use them effectively.

---

## Dashboard Components

### 1. Metrics Overview

The top section displays eight key performance indicators (KPIs) that provide a quick snapshot of organizational health:

| Metric | Description | Target |
|--------|-------------|--------|
| **Total Employees** | Number of active users on the platform | Increasing |
| **Avg FWI Score** | Average Financial Wellness Index (0-100) | > 70 |
| **Employees at Risk** | Count and percentage of at-risk employees | < 20% |
| **Intervention Success Rate** | Percentage of successful interventions | > 80% |
| **Active Interventions** | Number of ongoing intervention workflows | Varies |
| **Completed Interventions** | Total completed interventions | Increasing |
| **Churn Risk Average** | Average churn probability across organization | < 40% |
| **Engagement Score** | Overall user engagement metric | > 75 |

**Color Coding:**
- Green (75+): Healthy
- Yellow (50-74): Needs attention
- Red (<50): Critical

### 2. Overview Tab

#### Churn Risk Analysis Chart
Visualizes churn risk trends over the last 30 days. The area chart shows:
- **X-axis**: Date range (last 30 days)
- **Y-axis**: Churn risk percentage (0-100%)
- **Trend**: Red area indicates risk level

**How to use:**
- Monitor upward trends for early warning signs
- Identify seasonal patterns in churn risk
- Correlate spikes with specific events or campaigns

#### FWI Score Trend
Shows the evolution of average Financial Wellness Index over time.

**How to use:**
- Track improvement from interventions
- Identify departments or segments with declining scores
- Set targets for quarterly improvements

#### Employees at Risk Table
Lists the top 5 employees requiring immediate attention with:
- Name and email
- Current FWI Score
- Churn risk probability
- Risk level classification
- Number of active interventions
- Last activity timestamp

**Action Items:**
- Click "Intervene" button to launch intervention workflow
- Review last activity to understand engagement patterns
- Track intervention history for each employee

## Campaigns Tab

### Creating a Push Notification Campaign

1. **Click "New Campaign" button**
2. **Fill in campaign details:**
   - **Campaign Name**: Descriptive name (e.g., "Q4 Engagement Boost")
   - **Campaign Type**: Select from predefined types:
     - Churn Alert: Alert users about churn risk
     - Intervention Offer: Present intervention opportunity
     - Engagement Boost: Encourage app usage
     - Achievement Unlock: Celebrate milestones
     - Goal Reminder: Remind about financial goals
     - Educational Content: Share learning resources
     - Promotional: Special offers

   - **Notification Title**: Short, engaging headline (max 50 chars)
   - **Message Body**: Detailed message (max 240 chars)
   - **Target Segment** (Optional): Specific user segment
   - **Target Risk Level** (Optional): Critical, High, Medium, Low

3. **Click "Create Campaign"**

### Campaign Analytics

Each campaign card displays:
- **Status Badge**: Draft, Scheduled, Sent, or Cancelled
- **Message Preview**: How notification appears to users
- **Metrics Grid:**
  - **Sent**: Total notifications delivered
  - **Opened**: Number of users who opened notification
  - **Clicked**: Number of users who clicked through
  - **Open Rate**: Percentage of sent that were opened
  - **Click Rate**: Percentage of sent that were clicked

When an intervention completes:
1. Navigate to Interventions tab
2. Find the completed intervention
3. Click "Record Success" button
4. Enter metrics:

   - Pre/post churn risk
   - Engagement increase
   - Estimated vs. actual savings

## Key Insights and Actions

### Red Flags (Immediate Action Required)

1. **Churn Risk > 70%**: Launch critical intervention immediately
2. **FWI Score < 30**: Employee in financial crisis
3. **No Activity > 7 days**: Engagement has dropped significantly
4. **Multiple Failed Interventions**: Employee needs different approach

### Green Indicators (Positive Trends)

1. **FWI Score Improving**: Interventions are working
2. **Engagement Increasing**: User is more active
3. **Churn Risk Declining**: Employee retention improving
4. **High Campaign Open Rates**: Messages are resonating

### Optimization Strategies

**For Churn Reduction:**
- Target high-risk employees with personalized offers
- Use manager alerts for critical cases
- Follow up with counseling for sustained improvement

**For Engagement:**
- Send achievement notifications to high performers
- Use goal reminders for goal-focused users
- Provide educational content to learners

**For FWI Improvement:**
- Combine counseling with personalized offers
- Use education for sustainable long-term gains
- Manager alerts for crisis intervention

## Reports and Exports

### Available Reports

1. **Monthly Summary**: All metrics and trends for the month

- Pause or resume as needed

## Troubleshooting

### Dashboard Not Loading

1. Clear browser cache
2. Refresh the page
3. Check internet connection
4. Verify you have admin/executive access

### Metrics Not Updating

1. Wait 5-10 minutes for data refresh
2. Check if data source is connected
3. Verify employee data is being collected
4. Contact IT support if issue persists

### Campaign Not Sending

1. Verify campaign is not in draft status
2. Check target segment has active users
3. Confirm device tokens are registered
4. Review campaign scheduling

## Frequently Asked Questions

**Q: How often are metrics updated?**
A: Metrics are updated every 5-10 minutes. Some real-time metrics update immediately.

**Q: Can I export campaign data?**
A: Yes, use the "Download" button on campaign cards to export analytics.

**Q: How do I target specific departments?**
A: Use the "Target Segment" field when creating campaigns or use the department filter in metrics.

**Q: What's a good FWI score?**
A: 70+ is healthy, 50-69 needs attention, below 50 is critical.

**Q: How long do interventions typically take?**
A: Most interventions run 14-90 days depending on type. See Intervention Types table.

**Q: Can I schedule campaigns in advance?**
A: Yes, use the "Scheduled At" field when creating campaigns.
