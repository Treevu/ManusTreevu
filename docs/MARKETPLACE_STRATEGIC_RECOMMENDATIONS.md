```

### 3.2 Pricing B2B (Marketplace)

```
OPCIÓN 1: REVENUE SHARE
├─ Treevü toma 10-15% de comisión de ventas adicionales
├─ Marketplace se beneficia de mayor volumen
├─ Vendedores pagan menos (comisión menor)
└─ Ejemplo: +40% ventas = +S/ 57,600/año para Treevü (10%)

OPCIÓN 2: SUBSCRIPTION + REVENUE SHARE
├─ Marketplace paga S/ 5,000-10,000/mes
├─ Treevü toma 5% de comisión de ventas adicionales
├─ Mejor para ambos
└─ Ejemplo: S/ 10K/mes + 5% de S/ 144K = S/ 17,200/mes

OPCIÓN 3: WHITE LABEL
├─ Marketplace integra Treevü como feature propia
├─ Treevü recibe S/ 20,000-50,000/mes
├─ Marketplace controla pricing
└─ Mejor para escala

PROYECCIÓN (1,000 vendedores activos):
├─ B2C Revenue: S/ 99-999/mes × 1,000 = S/ 99K-999K/mes
├─ B2B Revenue: S/ 5-50K/mes × 10 marketplaces = S/ 50-500K/mes
└─ TOTAL: S/ 149K-1.5M/mes = S/ 1.8M-18M/año
