## 2. OPORTUNIDADES IDENTIFICADAS

### 2.1 Pilar 1: Identificación de Compradores Listos para Comprar

**Problema:** Vendedor no sabe quién está realmente interesado.

**Solución:** Scoring de "Buyer Readiness" basado en salud financiera.

#### 2.1.1 Buyer Readiness Score

```
DIMENSIONES:

1. CAPACIDAD FINANCIERA (40% del peso)
   ├─ FWI Score (0-100)
   │  └─ FWI > 70 = Comprador con capacidad
   ├─ Ahorros disponibles
   │  └─ Ahorros > monto del producto = Listo
   ├─ Deuda actual
   │  └─ Deuda < 40% ingresos = Capacidad de más deuda
   └─ Ratio de capacidad de pago
      └─ Ingresos mensuales / Precio del producto

2. INTENCIÓN DE COMPRA (30% del peso)
   ├─ Cambios recientes en gastos
   │  └─ Aumento de gastos = Mayor capacidad
   ├─ Búsqueda de préstamos
   │  └─ Búsqueda activa = Urgencia
   ├─ Reducción de deuda
   │  └─ Preparación para compra grande
   ├─ Cambios de vida
   │  └─ Matrimonio, hijo, mudanza = Oportunidades
   └─ Estacionalidad
      └─ Bonificaciones, fin de año = Mayor capacidad

3. URGENCIA (20% del peso)
   ├─ Frecuencia de uso de EWA
   │  └─ Uso frecuente = Urgencia de capital
   ├─ Cambios recientes de empleo
   │  └─ Nuevo empleo = Mejor capacidad
   ├─ Aumento de ingresos
   │  └─ Más dinero = Mayor capacidad
   └─ Tiempo desde última compra similar
      └─ Ciclo de reemplazo

4. COMPATIBILIDAD (10% del peso)
   ├─ Gastos históricos en categoría
   │  └─ Si gasta en autos, busca autos
   ├─ Rango de precio preferido
   │  └─ Si gasta S/ 10K, busca S/ 10K
   └─ Patrón de compra
      └─ Frecuencia y monto

BUYER READINESS SCORE (0-100):
├─ 0-30: No listo (bajo interés, sin capacidad)
├─ 31-60: Potencial (interés pero sin capacidad)
├─ 61-80: Listo (capacidad y algún interés)
└─ 81-100: MUY LISTO (capacidad + urgencia + intención)

CASOS DE USO:
├─ Identificar compradores MUY LISTOS (81-100)
├─ Priorizar esfuerzo de venta
├─ Personalizar mensaje de venta
├─ Ofrecer opciones de financiamiento
└─ Aumentar tasa de cierre
```

### 2.2 Pilar 2: Análisis de Comportamiento de Compradores

**Problema:** Vendedor no entiende por qué comprador no cierra.

**Solución:** Dashboard que muestra comportamiento y patrones de compra.

#### 2.2.1 Análisis de Abandono

```
PROBLEMA: 70-80% de compradores interesados no cierran

CAUSAS IDENTIFICADAS:

1. INCERTIDUMBRE FINANCIERA (40%)
   ├─ Comprador no sabe si puede pagar
   ├─ Comprador no sabe si es buen momento
   ├─ Comprador tiene dudas sobre financiamiento
   └─ Solución: Mostrar opciones de pago claras

2. COMPARACIÓN CON COMPETENCIA (30%)
   ├─ Comprador está viendo otras opciones
   ├─ Comprador está comparando precios
   ├─ Comprador está viendo mejores condiciones
   └─ Solución: Diferenciarse + Urgencia

3. FALTA DE CONFIANZA (15%)
   ├─ Comprador desconfía del vendedor
   ├─ Comprador desconfía del producto
   ├─ Comprador no ve suficiente información
   └─ Solución: Más información + Testimonios

4. TIMING INCORRECTO (10%)
   ├─ Comprador no está listo aún
   ├─ Comprador espera mejor momento
   ├─ Comprador espera más dinero
   └─ Solución: Nurture + Seguimiento

5. OTROS (5%)
   ├─ Comprador cambió de opinión
   ├─ Comprador encontró mejor opción
   └─ Solución: Retargeting

TREEVÜ PUEDE IDENTIFICAR CAUSA POR COMPRADOR
→ PERSONALIZAR ESTRATEGIA DE CIERRE
```

### 2.4 Pilar 4: Predicción de Demanda

**Problema:** Vendedor no sabe qué va a vender.

**Solución:** Predicción de demanda basada en patrones de compra.

#### 2.4.1 Predicción de Demanda por Categoría

```
MODELO: Análisis de patrones históricos + Salud financiera actual

INPUTS:
├─ Histórico de compras por categoría
├─ Estacionalidad (bonificaciones, fin de año)
├─ Cambios de vida (matrimonio, hijo, mudanza)
├─ Cambios económicos (inflación, desempleo)
├─ Cambios de salud financiera (FWI, deuda)
└─ Ciclo de reemplazo (auto cada 5 años, etc.)

OUTPUTS:
├─ Demanda esperada (próximos 30, 60, 90 días)
├─ Segmentos con mayor demanda
├─ Precios esperados
├─ Volumen esperado
└─ Recomendaciones de inventario

EJEMPLO: MERCADO DE AUTOS

Análisis histórico:
├─ Enero: +30% (bonificaciones)
├─ Febrero-Marzo: +15% (recuperación)
├─ Abril-Agosto: -10% (baja estacional)
├─ Septiembre: +20% (gastos escolares, búsqueda)
├─ Octubre-Noviembre: +25% (búsqueda de fin de año)
└─ Diciembre: -5% (presupuesto limitado)

Análisis de FWI:
├─ Enero 2024: FWI promedio 52 (capacidad media)
├─ Febrero 2024: FWI promedio 48 (capacidad baja)
├─ Marzo 2024: FWI promedio 55 (capacidad media)
└─ Tendencia: Mejorando

Predicción para Marzo 2024:
├─ Demanda esperada: +15% vs promedio
├─ Compradores listos: 15-20 (vs 10-12 normal)
├─ Precio esperado: S/ 52,000-58,000
├─ Volumen esperado: 8-10 unidades
└─ Recomendación: Tener 12-15 unidades en stock

IMPACTO: +20-30% en eficiencia de inventario
```

├─ Marketplace integra Treevü como feature propia
├─ Treevü recibe S/ 20,000-50,000/mes

└─ Mejor para escala
