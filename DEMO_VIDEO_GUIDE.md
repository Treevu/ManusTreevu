# Guía para Video de Respaldo - Demo Day Treevü

## Propósito
Este video sirve como respaldo en caso de problemas técnicos durante la demostración en vivo.

## Duración Recomendada
3-5 minutos máximo

## Escenas a Grabar

### Escena 1: Landing Page (15 segundos)
- Mostrar la página principal con el mensaje "El Sistema Operativo"
- Hacer scroll suave para mostrar las secciones de beneficios
- Destacar el botón "Calcular mi ROI"

### Escena 2: Calculadora ROI (30 segundos)
- Ingresar datos de ejemplo:
  - 500 empleados
  - Salario promedio: $15,000 MXN
  - Ausentismo actual: 8%
- Mostrar los resultados del cálculo
- Destacar el ahorro proyectado

### Escena 3: Dashboard Empleado (60 segundos)
1. **FWI Score** - Mostrar el indicador de bienestar financiero
2. **EWA Slider** - Demostrar solicitud de adelanto:
   - Mover el slider a $2,000
   - Mostrar el cálculo de comisión
   - Confirmar la solicitud
3. **TreePoints** - Mostrar el balance y leaderboard
4. **Metas de Ahorro** - Crear una meta rápida

### Escena 4: Dashboard B2B Admin (45 segundos)
1. **Torre de Control** - Mostrar métricas agregadas
2. **Alertas** - Mostrar sistema de alertas por departamento
3. **Reportes** - Generar un reporte ejecutivo
4. **Pulse Survey** - Mostrar encuestas de bienestar

### Escena 5: Dashboard Merchant (30 segundos)
1. **Ofertas Activas** - Mostrar lista de ofertas
2. **Validación QR** - Simular escaneo de cupón
3. **Analytics** - Mostrar métricas de redención

### Escena 6: Gamificación (30 segundos)
1. **Badges** - Mostrar colección de insignias
2. **Leaderboard** - Mostrar ranking de empleados
3. **Educación** - Completar un tutorial corto

## Consejos de Grabación

### Antes de Grabar
- [ ] Ejecutar `npx tsx scripts/seed-demo.ts` para datos frescos
- [ ] Limpiar caché del navegador
- [ ] Usar resolución 1920x1080
- [ ] Desactivar notificaciones del sistema
- [ ] Usar un usuario de prueba con datos completos

### Durante la Grabación
- Movimientos de mouse suaves y deliberados
- Pausar 2 segundos en cada pantalla importante
- No hacer clicks muy rápidos
- Mostrar tooltips cuando sea relevante

### Herramientas Recomendadas
- **OBS Studio** - Grabación gratuita
- **Loom** - Grabación rápida con narración
- **QuickTime** (Mac) - Grabación nativa
